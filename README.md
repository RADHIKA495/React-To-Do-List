MUI React Todo List
A clean, responsive, and persistent Todo application built with React and Material UI. This project features a modern user interface with full CRUD (Create, Read, Update, Delete) capabilities and local storage persistence.

##   Features


- Add Tasks: Quick entry via a centered, styled input form.
- Persistent Storage: Uses localStorage to save your todos so they remain after a page refresh.
- Toggle Completion: Click any task to mark it as completed using a controlled checkbox.
- Remove Tasks: Delete items instantly using the Material UI DeleteIcon button.
- Responsive Design: Built with Material UI components like AppBar, List, and Box for a seamless experience across screen sizes.
- Unique IDs: Leverages crypto.randomUUID() for robust, unique key management.
